# Online Movie Registration Form

A simple static registration form for a movie streaming account. The form collects basic user details, checks required fields in the browser, and shows a confirmation page after submission.

## Features

- Registration form for basic account details
- Gender, phone, email, password, genre, and plan fields
- Browser validation for required fields and matching passwords
- Confirmation page after successful submission
- Responsive styling for desktop and mobile screens

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Setup

No build tools are required.

Open `src/index.html` in a browser and submit the form to view the confirmation page.
